What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to a part of the document, such as elements, attributes, or text.

What is event delegation in the context of the DOM , and Why is it usefull?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.
It is usefull because event delegation allows you to attach a single listener to the parent container.
